# Project 2: Command Line Shell

See: https://www.cs.usfca.edu/~mmalensek/cs326/assignments/project-2.html

Author: Ramneet Kaur

About project: 
In this project, we implemented a shell of our own, starting out with naming it. It starts with my shell and will print its prompt and wait for user input. The shell also displays information about the prompt while also using scripting, built-in commands, signal handling,I/O redirection, background jobs, incremental history search, tab completion and printing the history.

About Shell:
The shell is in interactive mode, doing actions that are described in the tests.

Included Files:
There are several files included. They are:
Makefile
history.c
history.h
logger.h
shell.c
ui.c
ui.h

To compile and run this shell:
make ./rosh

Testing:
To execute test cases, use make test.
To pull updated test cases, use make testupdate.
To view a specific test, type that test number.
